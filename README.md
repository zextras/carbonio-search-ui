<h3>How to build</h3>

<h4>Setup</h4>

- clone this repo

- install the dependencies:
```
npm install
```

<h4>Watch Mode</h4>

```
npm run start -- -h <host>
```

The host parameter is required to proxy requests and content from an existing Carbonio installation.

<h4>Build</h4>

```
npm run build
```
